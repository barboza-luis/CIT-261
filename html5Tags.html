<< HTML5 Tags - Video, Audio, and Canvas.

Before HTML5, a video, audio or drawing could only be displayed in a browser through a format, an extension or plug-in. It was then standardized and included across major browsers in the introduction of HTML5.

<video> element specifies a standard way to embed a video in a web page.

<audio> element specifies a standard way to embed audio in a web page.

<canvas> element is used to draw graphics, on the fly, via scripting (usually JavaScript). This element is only a container for graphics, Similar to a blank canvas that a painter works off. Canvas has several methods for drawing paths, boxes, circles, text, and adding images.
